# 📊 청소년 범죄율에 영향을 미치는 요소를 고려하여 향후 1년간의 지역별 청소년 범죄율 예측

이 프로젝트는 전국 시도별 청소년 범죄율과 다양한 관련 요인을 분석하여 예측하는 데이터 기반 프로젝트입니다. Python과 데이터 시각화 도구를 활용해 2013년부터 2022년까지의 데이터를 분석하였으며, 향후 청소년 범죄율 감소에 기여하는 것을 목표로 합니다.
